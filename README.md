# FinOps Toolkit
A collection of scripts, automation, and tools for AWS cost optimisation.

## 📁 Structure
- `audits/`: AWS waste detection and 8-pattern discovery.
- `automation/`: Lambda functions for auto-stopping and cleanup.
- `scripts/`: Boto3 utilities and CLI helpers.
- `calculators/`: TCO and Savings Plan modeling.
- `habits/`: Checklists for operational excellence.
